# Datenschutzerklärung – Wickelfinder

*Stand: 2026-07-12*

Wickelfinder ist so gebaut, dass so wenig Daten wie möglich anfallen.

## Welche Daten werden verarbeitet?

- **Standort:** Dein Standort wird ausschließlich **auf dem Gerät** verwendet,
  um Wickelplätze in deiner Nähe anzuzeigen. Er wird **nicht** an einen Server
  von Wickelfinder übertragen und **nicht** gespeichert.
- **Kartendaten:** Die App lädt Kartenkacheln und Wickelplatz-Daten von
  OpenStreetMap-Servern. Dabei wird – wie bei jedem Internetdienst – technisch
  deine IP-Adresse an diese Server übermittelt. Es gelten die
  Datenschutzbestimmungen von [OpenStreetMap](https://wiki.osmfoundation.org/wiki/Privacy_Policy).

## Was wir NICHT tun

- ❌ Kein Tracking, keine Analyse-SDKs
- ❌ Keine Werbung
- ❌ Keine Benutzerkonten mit Klarnamen
- ❌ Kein Verkauf oder Weitergabe von Daten

## Community-Funktionen (geplant)

Sobald das Hinzufügen und Bewerten von Plätzen verfügbar ist, erfolgt dies
über **anonyme Authentifizierung** (kein Name, keine E-Mail erforderlich).
Beigetragene Platzdaten (Koordinaten, Bewertung) sind öffentlich sichtbar,
enthalten aber keine personenbezogenen Daten. Diese Erklärung wird dann
entsprechend ergänzt.

## Kontakt
